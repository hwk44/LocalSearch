### 검색알고리즘
    검색은 대부분 정렬을 동반함.
    정렬은 메모리를 필요로 함
    메모리를 적게 쓰는 데에 집중

    이진 검색(가장 대표적)
    정렬을 동반하지 않는 검색문제 N.P(대표적) => Local
    제한된 조건(Local) 하에서 풀림 => 조건을 어떻게 설정할 것인가?

### Hill Climb Algorithm
    특정 좌표(x,y)를 기준으로 가장 낮/높은 값을 찾아라
    좌표를 기반으로 좌우 값을 좀 더 살펴보자 => 계산이 오래 걸림
